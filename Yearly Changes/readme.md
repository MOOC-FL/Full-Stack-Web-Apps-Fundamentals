#### No more yearly versions
- There are no "annual versions" of the course, the course is open all the time. Each part is updated once or twice per year. Updates are Mostly minor: library versions are updated and text clarity is improved. However, there might also be some bigger changes.
- Despite changes all the submitted exercises remain valid , and the course can be continued without being bothered about updates. Also, the policy for getting certificates, university credits etc. shall remain the same no matter what happens.

#### Course Versioning Policy

| Aspect | Detail |
|--------|--------|
| **Annual versions** | None — course is open all the time |
| **Update frequency** | Each part updated 1–2 times per year |
| **Typical changes** | Library version updates, text clarity improvements |
| **Occasional changes** | Larger structural or content updates |
| **Exercise validity** | All submitted exercises remain valid despite changes |
| **Continuation** | Course can be continued without being bothered about updates |
| **Policies** | Certificates, university credits, etc. remain the same regardless of updates |
